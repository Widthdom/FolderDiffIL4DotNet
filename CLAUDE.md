# CLAUDE.md

## English

Claude Code should use `AGENT_GUIDE.md` as the shared source of truth for this repository.

Before making changes:

1. Read `AGENT_GUIDE.md`.
2. Follow the global `cdidx` code-search policy.
3. Use `.codex/workflows/` for task-specific workflows when relevant.
4. Do not create tags or publish packages.
5. Keep English and Japanese documentation consistent when both are present.

Do not duplicate long instructions here. Update `AGENT_GUIDE.md` instead.

## 日本語

Claude Code では、このリポジトリの共通ルールは `AGENT_GUIDE.md` を正本として扱ってください。

変更前に次を確認してください。

1. `AGENT_GUIDE.md` を読む。
2. グローバル `cdidx` のコード検索ポリシーに従う。
3. 必要に応じて `.codex/workflows/` の手順を使う。
4. タグ作成やパッケージ公開はしない。
5. 英語版と日本語版があるドキュメントは内容を揃える。

長い指示はここに重複させず、`AGENT_GUIDE.md` を更新してください。
